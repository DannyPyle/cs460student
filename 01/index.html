RO.ME "3 DREAMS OF BLACK"

RO.ME is a music video that is interactive. RO.ME uses a technology called WebGL. It uses harware acceleration by using the users gpu to render models through a supporting browser like chrome. When the user moves the mouse, models are rendered which follow the direction of the mouse. The impact that RO.ME had was great because it was one of the first examples of using webGL to create graphics in a browser using the gpu of the computer running the program. This opens the door to have more interactive experieces through a browser. (games, music videos like this etc.)

Technology: the technology used to create RO.ME is webGL. WebGL is an 'extension of javaScript' that is 'an extension of the browser using html and javascript'.
